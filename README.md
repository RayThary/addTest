# Unity Addressables Remote Load Test

Unity에서 GitHub Pages를 활용해 Addressables 원격 리소스 로드 경로를 테스트하기 위한 저장소입니다.

## 테스트 목적
- GitHub Pages를 활용한 정적 파일 호스팅 테스트
- Addressables Remote Load Path 설정 테스트
- catalog.json / catalog.hash / bundle 파일 업로드 및 경로 확인
- Unity에서 원격 카탈로그와 번들 파일을 불러오는 흐름 확인

## Note
실제 게임 서버나 DB를 구현한 프로젝트는 아니며, Unity Addressables의 원격 리소스 다운로드 구조를 학습하기 위한 테스트 저장소입니다.
